# hello-kubernetes
A kubernetes "Hello world" for DevopsLisbon meetup.

```
├── Dockerfile        - Container recipy for the NodeJS hello world server
├── demo_next         - sourcing this file will echo the next command to be executed the terminal.
├── demo.sh           - All the commands that will be run as part of the demo
├── server.js         - The node js code that will run inside the container
├── setup.sh          - download kubernetes and configure minikube 
├── server_with_ip.js - improved version that will print the bound network interface
└── simple_server.js  - server that just prints hello world.
```
